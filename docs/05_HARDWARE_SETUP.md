## 카메라 위치

### 정면 카메라 (SN: 021222070391)
- **위치**: 대시보드 중앙 또는 룸미러 근처
- **각도**: 운전자 얼굴 정면
- **확인**: 카메라에 운전자 얼굴 전체가 보이는지

### 측면 카메라 (SN: 405622073483)
- **위치**: 조수석 쪽 또는 A필러
- **각도**: 운전자 상반신 (얼굴 + 어깨 + 손)
- **확인**: 핸들 잡은 손이 보이는지

## USB 허브 주의사항

> ⚠️ **외부 전원이 있는 USB 허브를 사용해야 합니다!**
> 전원 없는 허브를 쓰면 카메라/동글에 전력 부족으로 끊김이 발생합니다.

- 허브 모델: Realtek RTS5411 (또는 동급)
- 허브 전원 어댑터: **반드시 연결**
- LED가 켜져있는지 확인

## 워치 준비

### 실험 전
1. 충전 크래들에서 **빼기**
2. 화면 **터치해서 깨우기**
3. 피실험자 **왼쪽 손목**에 채우기
4. 밴드를 **적당히 조이기** (너무 느슨하면 PPG 안 나옴)

### 실험 후
1. 피실험자에게서 **빼기**
2. 충전 크래들에 **꽂기** (다음 실험까지 충전)

### 배터리
- 완충 시 약 8시간 연속 사용 가능
- 하루 실험이면 아침에 빼서 저녁까지 OK
- 2일 이상이면 매일 밤 충전

## RODE 마이크 준비

### TX (송신기 - 피실험자에게)
1. 전원 버튼 길게 눌러서 **켜기** (LED 빛남)
2. 피실험자 **옷 앞쪽에 클립**으로 고정
3. 마이크 구멍이 **입 방향**을 향하게

### RX (수신기 - Jetson에 연결)
- USB-C로 Jetson/허브에 연결
- 전원은 USB에서 공급 (별도 조작 불필요)

### TX 내장 녹음 (백업)
- TX는 자체적으로도 녹음합니다 (내장 메모리)
- 메인 녹음이 실패해도 TX에 백업이 있음
- 백업 꺼내는 법:
  1. RX **먼저 끄기**
  2. TX도 **끄기**
  3. TX를 USB-C로 Jetson에 직접 연결 (전원 버튼 누르지 마세요!)
  4. 자동으로 USB 메모리로 인식됨
  5. `lsblk` → `/dev/sdX` 확인 → mount해서 WAV 복사

## 전원 켜는 순서 (처음 세팅 시)

1. Jetson 전원 연결 → 켜기
2. 바탕화면 나올 때까지 기다리기 (1~2분)
3. USB 허브 전원 연결
4. USB 허브를 Jetson에 꽂기
5. 카메라, 동글, RODE RX 꽂기
6. 10초 기다리기
7. 장치 확인:
```bash
ls /dev/ttyACM*     # 동글
ls /dev/video*      # 카메라
lsusb               # 전체 USB
```

---

## 전원 끄는 순서

1. 센싱 종료: `./ops/stop.sh`
2. 데이터 검증: `python3 validate/validate_session.py data/C0XX`
3. 워치를 충전 크래들에 꽂기
4. RODE TX 끄기
5. Jetson 종료: `sudo shutdown now`
6. 전원 어댑터 분리
