K-MER 센싱 시스템

차량 시뮬레이터 멀티모달 데이터 수집 시스템 — v2.0 (2026-04-01)

센싱 시작하기

1 워치 준비
충전 크래들에서 빼기 → 옆면 버튼 1초 꾹 눌러서 깨우기 → 피실험자 손목에 채우기
2 마이크 준비
RODE TX 전원 켜기 (LED 확인) → 피실험자 옷에 클립 고정
3 센싱 시작
바탕화면 센싱 시작 아이콘 더블클릭
참가자 ID 확인 → 초록 버튼 클릭 → 40초 기다리기
4 확인
LED가 전부 초록이면 성공! 실험 진행.
터미널로 시작하려면:
cd ~/work/sensing-collector && bash ops/start.sh C001

버튼 기능

센싱 시작

참가자 ID 입력 후 클릭.
자동으로: USB 확인 → 동글 리셋 → 센싱 → 모니터링 → 대시보드 → 백업
센싱 종료

실험 끝나면 클릭.
(저녁 7시에 자동 종료됨)
문제해결

센서에 빨간불이 켜지면 클릭.
단계별로 안내해줍니다:
워치 확인 → 동글 리셋 → 워치 재연결 → USB 리셋 → Claude 진단
매뉴얼

사용법 팝업 표시.

문제가 생겼을 때

가장 중요한 규칙: 영상이 돌고 있으면 절대 건드리지 마세요!
워치만 문제인 경우가 99%이고, 자동으로 복구됩니다.

자동 복구 시스템

모니터링 프로그램이 2분마다 워치 상태를 확인합니다.

워치가 끊기면 자동으로: 동글 리셋 → 워치 재연결 (영상/음성은 계속 녹화)

대부분의 문제는 2~3분 안에 자동으로 해결됩니다.

자동 복구가 안 될 때 (직접 해결)

1 2분 기다려보세요. 자동 복구가 시도됩니다.
2 [문제해결] 버튼을 클릭하세요. 단계별로 안내해줍니다.
3 그래도 안 되면? 준영이에게 연락하세요. 원격으로 해결할 수 있습니다.

흔한 문제와 원인

증상원인해결
PPG/GSR 빨간불 워치가 잠들어서 연결 끊김 자동 복구 대기 or [문제해결] 클릭
전부 빨간불 USB 허브 전원 빠짐 허브 전원 케이블 확인
음성 빨간불 RODE TX 전원 꺼짐 RODE TX 전원 버튼 누르기
영상 빨간불 카메라 USB 빠짐 카메라 USB 케이블 확인
시작 자체가 안 됨 디스크 꽉 참 or USB 문제 [문제해결] 클릭

실험 중

해도 되는 것

  • 모니터 화면 보기
  • LED 상태 확인하기
  • 마우스로 다른 창 열기

하면 안 되는 것

  • USB 케이블 건드리기
  • 터미널에서 Ctrl+C 누르기
  • 컴퓨터 끄기/재시작
  • 다른 프로그램 설치하기

실험 종료

1 센싱 종료
센싱 종료 버튼 클릭
(저녁 7시에 자동 종료되므로, 7시 전에 끝나면 직접 클릭)
2 워치 크래들에 꽂기
피실험자에게서 워치를 회수하고 충전 크래들에 꽂아두세요.
3 다음 참가자 시작 (있으면)
새 워치 채우기 → 화면 터치 → 센싱 시작

하루 일과 요약

시간할 일
실험 전워치 충전 확인, 크래들에서 빼기, 화면 터치
실험 시작[센싱 시작] 버튼 클릭
실험 중아무것도 안 해도 됨 (자동 모니터링)
참가자 전환[센싱 종료] → 워치 교체 → [센싱 시작]
마지막 실험 후[센싱 종료], 워치 크래들에 꽂기
19:00자동 종료 + 자동 백업 (아무것도 안 해도 됨)
