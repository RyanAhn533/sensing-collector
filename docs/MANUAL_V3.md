# 센싱 시스템 사용법

---

## 이건 꼭 알아두세요!

워치는 데이터를 **워치 안에 자동으로 저장**합니다.
화면에 "BLE 끊김"이 뜨더라도 **데이터는 안전**합니다.
당황하지 마세요. 워치가 알아서 저장하고 있어요.

실험이 끝나면 **워치를 충전 거치대(크레들)에 올려놓고 다운로드**하면 됩니다.

---

## 실험 시작하기

### 시작 전에 확인할 것
- 워치 충전 됐나? (화면에서 배터리 확인)
- 워치 화면에 블루투스 표시(B) 보이나?
- 워치를 피실험자 손목에 채웠나?
- 마이크(RODE) 송신기 전원 켰나?
- USB 허브 전원 연결 했나?

### 시작 방법
1. 화면에서 참가자 번호 확인하세요 (자동으로 다음 번호가 입력돼 있어요)
2. 초록색 **"센싱 시작"** 버튼을 누르세요
3. 40초 정도 기다리세요
4. 화면에 이렇게 뜨면 성공이에요:
   - **내장저장: 기록 중** (초록 불)
   - **배터리: OO%** (초록 불)

---

## 실험 도중에는 이것만 확인하세요

화면 오른쪽에 워치 상태가 보여요.

**초록 불이면 정상이에요. 아무것도 안 해도 됩니다.**

| 화면에 이렇게 뜨면 | 뜻 | 해야 할 것 |
|---|---|---|
| 내장저장 초록, BLE 초록 | 완벽하게 정상 | 그냥 두세요 |
| 내장저장 초록, BLE 노랑 "끊김" | 정상이에요! | 데이터는 저장되고 있어요. 그냥 두세요 |
| 내장저장 빨강 | 문제 있음 | 옆에 "해결법" 버튼 누르세요 |
| 배터리 빨강 | 배터리 부족 | 옆에 "해결법" 버튼 누르세요 |

**"BLE 끊김"은 문제가 아니에요!** 워치가 알아서 저장하고 있으니까 걱정 마세요.

---

## 실험 끝내기

1. 빨간색 **"센싱 종료"** 버튼을 누르세요
2. "크레들에 올려서 다운로드하세요" 메시지가 나와요

### 데이터 가져오기 (꼭 해주세요!)

1. 피실험자에게서 워치를 벗기세요
2. **충전 거치대(크레들)에 워치를 올려놓으세요**
3. 화면에서 보라색 **"데이터 다운로드"** 버튼을 누르세요
4. 자동으로 워치 안의 데이터를 컴퓨터로 가져와요
5. "다운로드 완료" 메시지가 나오면 끝!

**주의: 다운로드 안 하고 다음 실험 시작하면 이전 데이터가 날아갈 수 있어요!**

---

## 문제가 생겼을 때

화면에 문제가 생기면 **"해결법" 버튼**이 자동으로 나타나요.
그 버튼을 누르면 어떻게 해야 하는지 알려줘요.

혹시 "해결법" 버튼이 안 보이면 아래를 따라하세요.

### "워치를 깨워주세요"라고 뜰 때

워치가 잠들어 있어요.

1. 워치 옆면에 버튼이 2개 있어요. **아래쪽 버튼**을 1초 꾹 누르세요
2. 화면이 밝아지면서 시간이 보이면 성공
3. 블루투스 표시(B)가 보이는지 확인하세요
4. 30초 정도 기다리면 자동으로 연결돼요

### "내장저장: 기록 안 됨"이라고 뜰 때

1. "센싱 종료" 버튼을 누르세요
2. 5초 기다리세요
3. "센싱 시작" 버튼을 다시 누르세요
4. 40초 후에 "기록 중"으로 바뀌면 해결!

### 아예 시작이 안 될 때

하나씩 해보세요:

1. USB 허브에 파란색 작은 USB(동글)가 꽂혀 있나 확인
2. 워치 화면이 켜지나 확인 (안 켜지면 크레들에 올려서 충전)
3. 프로그램을 X로 닫고 바탕화면에서 다시 열기
4. 그래도 안 되면 컴퓨터 재부팅

### 배터리 부족 경고가 뜰 때

- 실험 중이면: 끝날 때까지 괜찮아요 (10%여도 1시간은 돼요)
- 실험 안 할 때: 크레들에 올려서 충전하세요

### 다운로드가 안 될 때

1. 크레들에 워치가 제대로 올라가 있나 확인 (살짝 눌러서 딱 맞게)
2. 크레들 USB 케이블이 컴퓨터에 꽂혀 있나 확인
3. 10초 기다렸다가 다시 시도
4. 안 되면 컴퓨터 재부팅 후 다시 시도

---

## 절대 하면 안 되는 것

1. 실험 중에 USB 케이블이나 허브 건드리기
2. 터미널(검은 화면)에서 직접 뭔가 입력하기
3. "BLE 끊김"이라고 당황해서 센싱 종료 누르기 (데이터는 안전해요!)
4. 다운로드 안 하고 다음 실험 바로 시작하기
